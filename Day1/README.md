# Band Name Generator

A simple command-line Band Name Generator built with Python as part of the 100 Days of Python Bootcamp.

## Description

This is an interactive program that generates a creative band name by combining two user inputs: the city you grew up in and your pet's name. The program demonstrates basic Python concepts including input handling and string concatenation.

## Features

- Interactive user prompts
- Simple string concatenation
- User-friendly interface

## How to Play

1. Run the script: `python P001_BandName_generator.py`
2. Answer the first question: Enter the name of the city you grew up in
3. Answer the second question: Enter your pet's name
4. See your generated band name!

## Requirements

- Python 3.x
- No external dependencies (uses only built-in functions)

```

What's name of the city you grew up in?
```
