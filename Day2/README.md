# Tip Calculator

A simple command-line Tip Calculator built with Python as part of the 100 Days of Python Bootcamp.

## Description

This is an interactive tip calculator that helps you split a restaurant bill among multiple people. The program calculates the total bill including tip and divides it equally among all participants, making it easy to split expenses fairly.

## Features

- Calculates total bill with tip percentage
- Splits bill evenly among multiple people
- Rounds the final amount to 2 decimal places
- Supports common tip percentages (10, 12, or 15%)

## How to Play

1. Run the script: `python P002_Calculator_Tip.py`
2. Enter the total bill amount (as a decimal number)
3. Enter the tip percentage you want to give (10, 12, or 15)
4. Enter the number of people to split the bill
5. See how much each person should pay!

## Requirements

- Python 3.x
- No external dependencies (uses only built-in functions)
