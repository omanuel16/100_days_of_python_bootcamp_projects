# Rock Paper Scissors Game

A simple command-line Rock Paper Scissors game built with Python as part of the 100 Days of Python Bootcamp.

## Description

This is an interactive Rock Paper Scissors game where you play against the computer. The game displays ASCII art for each choice and determines the winner based on classic rules:
- Rock crushes Scissors
- Paper covers Rock
- Scissors cut Paper

## Features

- Interactive gameplay with ASCII art visuals
- Input validation for user choices
- Clear win/loss/draw messages
- Random computer opponent

## How to Play

1. Run the script: `python Project_PaperRockScissors.py`
2. Choose your option:
   - Type `0` for Rock
   - Type `1` for Paper
   - Type `2` for Scissors
3. The computer will randomly choose its option
4. See who wins!

## Requirements

- Python 3.x
- No external dependencies (uses only built-in `random` module)
